auto minilua::details::FunctionName::identifier ( ) -> std::vector<Identifier>

This method looks through the function name and returns Identifiers in order they reference on tables e.g. the vector [id_1,id_2] refers to a function name "id_1.id_2" in lua syntax

◆ method()

auto minilua::details::FunctionName::method ( ) -> std::optional<Identifier>

The identifier returned here is the last one if it is present e.g. identifiers() returns [id_1,id_2] and method returns id_5 then the full functionname is: "id_1.id_2:id_5" in lua syntax

Returns
an empty optional if the function is not a method the function-/method- name if the function is a method
