Detailed Description

Represents a value in lua.

You can use most normal c++ operators on these value (+, -, *, /, []). If the operation can't be performed on the actual value type an exception will be thrown.

◆ force()

auto minilua::Value::force ( Value  new_value,
std::string  origin = "" 
) const -> std::optional<SourceChangeTree>

Forces this value to become 'new_value'. Does not actually change the value. This will only return a SourceChange that (when applied) would result in the this value being changed.

The return value should be returned in NativeFunctions otherwise this does not have an effect.

This throws an exception if the types of the values didn't match.
