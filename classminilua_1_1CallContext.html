◆ unary_numeric_arg_helper()

auto minilua::CallContext::unary_numeric_arg_helper ( ) const -> std::tuple<double, UnaryOrigin>

Convenience methods for writing functions one or two numeric arguments that should track the origin (e.g. sqrt or pow).

Usage:

auto [arg, origin] = ctx.unary_numeric_arg_helper();
origin.reverse = unary_num_reverse(...);
// ...
auto [arg1, arg2, origin] = ctx.binary_numeric_arg_helper();
origin.reverse = binary_num_reverse(...);
// ...
